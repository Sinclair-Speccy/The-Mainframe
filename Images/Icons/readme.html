From the Artist

June, 1998

The Apple Desktop CPU Collection by Corey Marion

If you need an icon representing the Apple desktop computer you are currently using you will find it in this collection. I have tried to make icons that represent every existing case design. Some machines used essentially the same case and I have tried to label them accordingly (i.e. the PowerMac 7200, 7300, 7600 and desktop G3 use the exact same case design, I labeled them the 7000/G3 Desktop series). if I have missed any please let me know.

This set includes all icons from my original Apple CPU Collection, all desktop computers from my Apple Expansion set, and a couple of new machines, namely the G3 minitower, the cool new G3 All-In-One which is only for the education market right now (maybe Apple will go the route they did with the e-mate and open it up to all channels), and the revolutionary iMac, the most individual computer since the original Macintosh.

I have also tweaked many of the icons to make them more accurate, especially the 500 series, the 5000 series and the original machines. For simplicity, I will continue to make updates to this set when new machines come out or when I find a better reference for one of the current icons and make improvements. Updates will be announced at the Iconfactory.

I would like to thank Jeremy Helm for an excellent reference for my Color Classic icon, and Dave Brasgalla for an excellent reference for the iMac.
