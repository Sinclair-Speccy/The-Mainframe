Welcome to The Mainframe!

The Mainframe is a place for me to upload my guides and reviews for people who are interested in mainly vintage technology, but this site caters to people who use modern technology and operating systems too. On the top are some links which can guide you around the site. This site was originally going to have a template and such made from scratch; however, I decided that doing so would take a lot of effort and time, so I decided to grab an already existing template. The site might end up much neater, but for now, take what I have to offer.

One thing I would like to mention is that this site isn't that great on mobile devices. Despite me being more of a mobile user than a PC one, trying to get the site to work on mobile did not go well and was buggy.

March 1st, 2024 Update: The long needed updates and more stuff!

Webmaster, the updates for your site aren't showing!

I guess you can say this is common with my site as it is hosted on GitHub Pages.

First, you should refresh the page with Ctrl + R if you are on Windows or Command + R if you are an Apple user. F5 usually works too.

If the first thing doesn't work for you, you should clear your browser's cache. GitHub uses CDN caching for pages, so there is a small delay for changes to be reflected on the actual page. If you don't know how to clear your browser's cache, check this site out (or you can look it up on a search engine if your browser isn't there and uses a different method).

As for mobile users, I guess refresh the page or close it and open it up again.
