Useful Links

The section of the site hosts links to useful places around the web which you can use for references or research. This page is kind of messy as it has bene resigned in a way that I like it with dropdown boxes that you can click on instead of having it all shoved in your face at once

This section has links to stuff related to operating systems and such.

  • English-Language Linux Distributions: List of some linux distros. I use this to find names of old distros to figure out their names, locate them and use
  • fsck.technology: Collection of OSes and software from various vendors. Has A/UX 0.7 source code
  • grumbeer.dyndns.org: Some distros. Appears to be old ones and has some like Blade and UBOS
  • pd.spuddy.org: Yggdrasil Linux CD-ROMs. Will need MS-DOS with CD-ROM support to view these and make a boot floppy with rawrite. Site also has some public domain CD images such as the 'Simtel-20 Archives' and 'The Complete GNU Software Library'
  • linux.mathematik.tu-darmstadt.de: Has some historic Linux distros such as Erposs and LST
  • tenox.pdp-11.ru/: Some operating system archive. Also has apps and documentation
  • SIMH: A simulator for historic computer systems. It also has software kits for OSes like DOS-15 and OS/8 and to be used with their simulator itself
  • CTO: Operating Systems: A list of some OSes. Not just Linux ones as it has redirects for microkernel-based OSes (YCX2, chaOS, etc) and Unix/UNIX-like OSes (Lites, THix, etc)
  • Server operating system technical comparison: Some other site for server OSes. Has info for OSes like BOS/360 and MCP
  • http://www.lugs.ch: A list of some Linux distros. Says Corel Linux is still active when it is not LOL
  • http://www.ossh.com: Some old Linux distros
  • www.linux-distros.com:Linux distro archive
  • freshmeat.sourceforge.net/tags/operating-systems: Freecode (formerly Freshmeat). A place to get mainly open-source software for programmers and developers. Is only in an archived state but has 483 projects tagged "Operating Systems", 307 projects tagged "Linux Distributions" and many more. As of 2022 and on this page redirects to SourceForge. I guess they punted the dormant state and made the link go somewhere else
  • Mac OS 9/7.6.1 for Windows/Mac OS 9.7.6.1/8 for OS X/macOS: A site to get a premade Mac OS 9 sheepshaver install for Windows, OS X and macOS. Has to be installed on a computer to run. It also has System 7.6.1 under Basilisk II for Windows, OS X and I believe macOS too. There is also Mac OS 8.
  • Virtual x86: x86 virtualization in your browser, recompiling x86 to wasm on the fly
  • Archimedes Live!: A web version of the Acron Archimedes that is based on the Arculator code to make it more accessible to people who wanted to try it out without having to download and configure an emulator.

Tech stuff! Well, more of computer systems and such. Some of these links redirect to retro computer ads and info pages so they can be useful if you want to do some research.

  • nosher.net: Site has some retro computer ads. The site says "A history of the microcomputer industry in 300 adverts".
  • mastodonpc.tripod.com: Someone's personal collection of computers. Has stuff like the Unisys 4336dx and Digital Venturis 5100.
  • DVQ.COM: Another site with a personal computer collection. It has mainframes, microcomputers and minicomputers. Also has retro computer ads.
  • Digibarn: Digibarn! They have all sorts of computers here and a lot of them too!
  • Computer Ads from the Past: It is what the site name says. Computer ads from the Past newsletter archive.
  • http://retro.hansotten.nl/: Tons of info on some early personal computers/SBC.

This section has links to useful emulators and ROM files in case your emulator needs them, like Sheepshaver. Some of the emulators are online while others you have to download.

Emulators
  • Mac OS 9/7.6.1 for Windows/Mac OS 9.7.6.1/8 for OS X/macOS: A site to get a premade Mac OS 9 sheepshaver install for Windows, OS X and macOS. Has to be installed on a computer to run. It also has System 7.6.1 under Basilisk II for Windows, OS X and I believe macOS too. There is also Mac OS 8.
  • Virtual x86: x86 virtualization in your browser, recompiling x86 to wasm on the fly
  • Archimedes Live!: A web version of the Acron Archimedes that is based on the Arculator code to make it more accessible to people who wanted to try it out without having to download and configure an emulator.
  • SIMH: A simulator for historic computer systems. It also has software kits for OSes like DOS-15 and OS/8 and to be used with their simulator itself
  • Cray J90 simulator: A simulator for the Cray J90 computer that can be accessed via ssh.
ROM files
  • Macintosh ROM Archive: A Macintosh ROM archive from the Internet Archive to be used in Basilisk, Mini vMac or SheepShaver.
  • Macintosh Garden Macintosh ROM archive: Macintosh ROM archive from Macintosh Garden that has ROMS dating from the earliest Macintosh 128k to the Old World G3 desktop "beige" Macintosh computers and Wallstreet PowerBooks plus a couple of Bandai Pippin ROMs are also included to be used in Basilisk, Mini vMac or SheepShaver. Has a ROM for the Quadra 800 which can be used to emulate A/UX in QEMU.
  • Macintosh Repository ROMS: Same as the other two sites above. Contains ROMS for the most popular Macintosh models ROM files for emulation purposes, ranging from the first 64K ROM from the Macintosh 128K to the 4 MB ROM files from the Bandai Pippin or PowerMac G3, listed below in ROM size, then by release date from oldest to newest.
  • Symbian OS ROMs: Symbian OS ROMs for the Nokia 5320, 5800, N95, E5, and N-Gage.
