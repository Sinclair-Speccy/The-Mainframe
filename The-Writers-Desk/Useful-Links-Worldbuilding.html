Useful links for worldbuilding

This section will contain links to sites I find useful when it comes to worldbuilding but as of now, there's not much here.

  • Supernatural Powers and Abilities: A Wikia wiki that has a list of supernatural powers and abilities.
  • Family Echo: A simple online website for making family trees. Can be saved is storage unless cookies and history are cleared. Trees can be exported as Gedcom files.
  • English to Shakespearean: An exaggerated English to Shakespearean translator
  • Bryn Donovan's writing inspiration and resources: actually found this blog by accident years ago but turns out to be really useful. Has all sorts of writing inspiration and resources.
  • The Tech Deck: A list of English interjections.
  • TOnline Symbolism Dictionary : A symbolism dictionary.
  • haracter Chart for Fiction Writers: This was given to my class in 2018 for a writing task where we had to make a fictional character. Found it really useful as it goes in-depth with how you should think about your characters.
  • The Internet Movie Script Database (IMSDb): Want an idea for dialogue from a movie but you can't remember how it went, or just want other ideas? The Internet Movie Script Database is a great resource not only for getting ideas but it is good for research too. The site lets you read or download movie scripts for free and in HTML format so you can read them right in your web browser.
  • Colorpedia: Colorpedia is a colour encyclopedia and aims to give human names for all colours.
  • 638 Primary Personality Traits: A list of 638 personality traits/

Tools and Generators

This section will contain links to tools and generators I use with worldbuilding.

  • Azgaar's Fantasy Map Generator: A web application generating interactive and highly customizable maps.
  • Fantasy name generators: My go-to site for getting all sorts of name ideas for fantasy stuff. Has a large collection of generators.
  • Days between 2 dates: A calculator for figuring out days between 2 dates. Can be useful for future-related stories.
  • How Old Was I on a Certain Day?: A calculator to calculate your age on a specific date.
